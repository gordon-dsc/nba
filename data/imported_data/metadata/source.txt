The data set in the import section was created by pulling nba all star names
and years of selection from a webistes various pages through the use of web 
scrapping. The website listed each game on a seperate page so every page in
the given date range of 1980-2022. Once a list of names where achieved the 
balldontlie api was used to generate the stats for every player in the year
of their selection.
